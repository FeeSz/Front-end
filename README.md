# 🚀 Front-end

Repositório contendo projetos desenvolvidos durante os estudos de Desenvolvimento Front-end, utilizando HTML, CSS e JavaScript. Os projetos abordam conceitos de responsividade, manipulação do DOM, estilização moderna e boas práticas de desenvolvimento web.

## 📚 Projetos Disponíveis

### 🎨 AC4 - InDecor
Landing Page para divulgação de um evento de design de interiores.

**Tecnologias:**
- HTML5
- CSS3

**Funcionalidades:**
- Página responsiva
- Formulário de inscrição
- Seções informativas sobre o evento
- Design moderno voltado para conversão de usuários

---

### ⚖️ Calculadora de IMC
Aplicação para cálculo do Índice de Massa Corporal.

**Tecnologias:**
- HTML5
- CSS3
- JavaScript

**Funcionalidades:**
- Cálculo automático do IMC
- Classificação do resultado
- Limpeza de formulário
- Exibição dinâmica dos resultados

---

### ⛏️ Minecraft Quiz
Quiz interativo inspirado no universo Minecraft.

**Tecnologias:**
- HTML5
- CSS3
- JavaScript

**Funcionalidades:**

- Resultado final ao término do quiz

## 🚀 Como Executar

### Clone o repositório

```bash
git clone https://github.com/FeeSz/Front-end.git
```

### Acesse a pasta

```bash
cd Front-end
```

### Execute qualquer projeto

Abra o arquivo `index.html` do projeto desejado em seu navegador ou utilize a extensão **Live Server** no Visual Studio Code.
